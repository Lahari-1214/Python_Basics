<h1> Calculator with History (Python Tkinter)</h1>
<pre>
A simple GUI-based calculator built using Python and Tkinter, which performs basic arithmetic operations and keeps a history of calculations.
This mini project is perfect for beginners to understand:
    1.Tkinter GUI basics
    2.Button events
    3.Exception handling
    4.Working with lists and widgets
</pre>

<h4> Features</h4>
<ul>
<li>
 Basic arithmetic operations: +, -, *, /

 Displays calculation history

 Clear current input

 Clear full history

 Error handling for invalid expressions

 User-friendly GUI </li></ul>
